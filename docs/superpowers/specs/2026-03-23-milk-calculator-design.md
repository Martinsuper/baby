# 奶量计算器输入组件设计

## 概述

将喂奶记录页面的奶量输入方式从普通数字输入框改为底部弹出式计算器，提升移动端输入体验。

## 交互流程

1. 用户点击奶量输入区域
2. 底部弹出计算器面板（带滑入动画）
3. 用户通过数字键盘输入奶量
4. 点击确认按钮后，面板收起，数值填入表单

## 视觉布局

```
┌─────────────────────────────┐
│        输入奶量              │
├─────────────────────────────┤
│                             │
│         120 ml              │  ← 大字体显示当前输入
│                             │
│  [90ml] [120ml] [150ml]     │  ← 历史记录标签（横向）
│                             │
├─────────────────────────────┤
│    1      2      3          │
│    4      5      6          │  ← 数字键盘
│    7      8      9          │
│   ←      0      ✓          │  ← 删除、0、确认
└─────────────────────────────┘
```

## 组件设计

### 新增组件: `MilkCalculator.vue`

**Props:**
- `modelValue: number | null` - 当前输入值
- `history: number[]` - 历史记录数组（最多3条）

**Emits:**
- `update:modelValue` - 更新当前值
- `confirm` - 确认输入
- `close` - 关闭计算器

**功能:**
- 数字键盘输入（0-9）
- 删除最后一位数字
- 确认当前值
- 历史记录快速选择

### 修改组件: `feeding-record.vue`

**改动:**
1. 奶量输入区域改为可点击按钮样式
2. 引入 MilkCalculator 组件
3. 添加 `amountHistory` 状态，维护最近3条历史记录
4. 点击奶量区域弹出计算器
5. 确认后更新表单值并保存历史记录

## 功能细节

### 数字键盘逻辑

- **数字键 0-9**: 追加数字到当前值末尾
- **删除键 (←)**: 删除当前值的最后一位
- **确认键 (✓)**: 确认当前值并关闭面板

### 输入限制

- 最大值 500，超过自动截断为 500
- 首位不能为 0（输入 0 后再输入数字，直接替换为该数字）
- 空值时点击确认不执行操作

### 历史记录

- 存储最近 3 条不同的奶量值
- 每次确认时，将新值添加到历史记录头部
- 重复值不重复添加，而是移到头部
- 无历史记录时隐藏历史区域

## 动画效果

- 面板弹出: `transform: translateY(100%)` → `translateY(0)`，时长 300ms
- 遮罩层: `opacity: 0` → `opacity: 1`，时长 300ms
- 点击遮罩可关闭面板

## 技术实现

- 使用 Vue 3 Composition API
- 使用 Pinia 或本地状态管理历史记录
- CSS 动画实现弹出效果
- 触摸事件优化移动端体验
